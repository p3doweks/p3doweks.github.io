Formy współpracy ​

Proponujemy kilka możliwych poziomów naszego zaangażowania.

  • Plan Konsultacje ​

    Aplikuj

    20-50h konsultacji

    Kluczowe elementy:

    • Audyt flow,
    • Wyznaczenie celów i inicjatyw,
    • "5 sekundowa" tablica kanban,
    • Wdrożenie limitu WIP,
    • CI/CD (testy) przechodzące w 5-10 min,
    • Poprawny deployment script,
    • Monitorowanie postępów

  • Plan Bootcamp ​

    Aplikuj

    od 10 do 25 dni szkoleniowych

    Warsztaty i szkolenia na Waszym kodzie:

    • Test-Driven Development (TDD) + Fail-fast,
    • Czyszczenie error logów via TDD (hackathon),
    • Domain-Driven Design (DDD) + Refactoring,
    • Trunk-Based Development + Feature toggles

  • Plan Infrastruktura ​

    Aplikuj

    od 40h dla prostych środowisk linuxowych

    Nasz zespół usprawnia Waszą infrastrukturę tak aby uzyskać:

    • stabilność,
    • samowystarczalność (self-service) programisty

    Dostarczymy:

    • Deployment script,
    • System monitoringu (via TDD) na stacku EFG lub Waszym,
    • Środowiska test/prod na żądanie (Terraform via TDD),
    • Single command development environment - jednodniowy onboarding pracownika

    Opcja:

    • Pair programming z Waszym inżynierem od infrastruktury
